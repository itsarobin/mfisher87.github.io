# My GitHub Pages user site

## Contributing

Please be very conscious about committing directly to the main branch, as that will
trigger an automated release and DOI. Use PRs generously!


## Requirements

* Quarto >= 1.3
* `quarto install tinytex`


## Build

```
cd content/
quarto render
```
